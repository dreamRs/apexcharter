# Paris Climate

Average temperature and precipitation in Paris for the period 1971-2000.

## Usage

``` r
climate_paris
```

## Format

A data frame with 12 observations and the following 3 variables:

- `month`:

  Month

- `temperature`:

  Temperature (in degree celsius).

- `precipitation`:

  Precipitation (in mm).
