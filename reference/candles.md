# Candlestick demo data

Candlestick demo data

## Usage

``` r
candles
```

## Format

A data frame with 60 observations and the following 5 variables:

- `datetime`:

  Timestamp.

- `open`:

  Open value.

- `high`:

  Highest value.

- `low`:

  Lowest value.

- `close`:

  Close value.
