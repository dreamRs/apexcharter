# Convert a `data.frame` to a `list`

Convert data to a format suitable for ApexCharts.js

## Usage

``` r
parse_df(data, add_names = FALSE)
```

## Arguments

- data:

  A `data.frame` or an object coercible to `data.frame`.

- add_names:

  Use names of columns in output. Can be logical to reuse `data` names
  or a character vector of new names.

## Value

A `list` that can be used to specify data in
[`ax_series`](https://dreamrs.github.io/apexcharter/reference/ax-series.md)
for example.
