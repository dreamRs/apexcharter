# eco2mix data

The dataset contains data about electricity consumption and production
in France between 2012 and 2022.

## Usage

``` r
eco2mix
```

## Format

A data frame with 3,033 observations and 3 variables.

## Source

Rte (Réseau et transport d'électricité)
(<https://www.rte-france.com/eco2mix> and
<https://opendata.reseaux-energies.fr/>)
